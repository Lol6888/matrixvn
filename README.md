# Matrix VN — Landing page

Trang giới thiệu dịch vụ liên lạc bảo mật cho tổ chức Việt Nam, xây dựng trên nền tảng
mã nguồn mở [Matrix](https://matrix.org) và ứng dụng [Element X](https://element.io).

## Nội dung

- `index.html` — toàn bộ trang trong một file duy nhất (HTML + CSS + JS inline, không phụ
  thuộc tài nguyên ngoài).

## Xem thử tại máy

```bash
python -m http.server 8377
# mở http://localhost:8377
```

## Triển khai

File tĩnh, deploy được ngay lên GitHub Pages, Vercel, Netlify hoặc bất kỳ hosting tĩnh nào —
chỉ cần trỏ về `index.html` ở thư mục gốc.

## Ghi chú

Dịch vụ triển khai độc lập, không trực thuộc và không đại diện cho The Matrix.org Foundation
hay Element (Vector Creations Limited). "Matrix" và "Element" là thương hiệu của các chủ sở
hữu tương ứng.
